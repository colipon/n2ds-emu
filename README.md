# Project Overview

The N2DS Emulation project aims to create a highly accurate and efficient emulator for the Nintendo 2DS gaming console, allowing users to experience Nintendo games on various platforms.

## Minimum Specifications
- **OS**: Windows 10 or later / macOS 10.15 or later / Linux (latest versions supported)
- **CPU**: Intel Core i5 or equivalent
- **RAM**: 8 GB minimum
- **GPU**: NVIDIA GeForce 960 or equivalent
- **Storage**: 1 GB free space for installation

## Setup Method
1. Clone the repository: `git clone https://github.com/colipon/n2ds-emu.git`
2. Navigate to the project directory: `cd n2ds-emu`
3. Install dependencies: `npm install` (or your preferred package manager)
4. Run the emulator: `npm start` (or your preferred start command)

## Branch Rules
- Use feature branches for new features (e.g., `feature/feature-name`)
- Bug fixes should be done in hotfix branches (e.g., `hotfix/issue-name`)
- All branches should be regularly merged into `main`.
